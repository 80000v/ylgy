# ylgy

羊了个羊 通关程序

## 使用

### 命令行

执行以下命令前，将 <your-custom-token> 替换为实际内容

```shell
ylgy --token <your-custom-token>
```

### Docker
执行以下命令前，将 `<your-custom-token>` 替换为实际内容
```shell
docker run --rm -it zc2638/ylgy sh -c 'ylgy --token <your-custom-token>' 
```
